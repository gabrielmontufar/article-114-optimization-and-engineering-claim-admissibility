# Third-party raw-data redistribution note

The supplementary package separates raw and processed data by redistribution status.

## Included raw data

The Zenodo steel-truss workbook is included in the submission ZIP at:

`external_raw_datasets/Zenodo_15658671_SteelTruss/LatentMechanisms_SteelTruss_ExperimentalData.xlsx`

It is included because the Zenodo record for DOI `10.5281/zenodo.15658671` declares a Creative Commons Attribution 4.0 International license. Reuse requires attribution to the original source.

## Not redistributed raw data

The raw DesignSafe/NEES SAC Steel Project files are not redistributed in the submission ZIP because an open redistribution license was not confirmed for the legacy NEES record used by the portal-frame response layer. The ZIP includes processed service-envelope points, the raw-data manifest, and the acquisition note needed to retrieve the public source externally.

The AISC Shapes Database spreadsheet is not redistributed in the submission ZIP because it is a third-party catalog resource. The ZIP includes the AISC audit outputs and an acquisition note explaining how a reviewer can obtain the official spreadsheet and rerun the optional catalog audit.

## Claim boundary

This redistribution decision does not change the scientific claim level. Included and non-included raw datasets both remain evidence-limited: the package supports published response checks and catalog-audit claims under declared assumptions, not safety approval, professional certification, or complete building-code approval.
