# Third-party raw-data redistribution note

The supplementary package separates processed response evidence from large or third-party raw source files.

## Not bundled raw data

The Zenodo steel-truss workbook `LatentMechanisms_SteelTruss_ExperimentalData.xlsx` is a public, citable source at DOI `10.5281/zenodo.15658671`, but it is not bundled in the submission ZIP. The ZIP instead includes processed response-support points, the raw-data manifest, and the acquisition note needed to obtain the workbook for optional raw-data reruns.

The raw DesignSafe/NEES SAC Steel Project files are not redistributed in the submission ZIP because an open redistribution license was not confirmed for the legacy NEES record used by the portal-frame response layer. The ZIP includes processed service-envelope points, the raw-data manifest, and the acquisition note needed to retrieve the public source externally.

The AISC Shapes Database spreadsheet is not redistributed in the submission ZIP because it is a third-party catalog resource. The ZIP includes the AISC audit outputs and an acquisition note explaining how a reviewer can obtain the official spreadsheet and rerun the optional catalog audit.

## Claim boundary

This redistribution decision does not change the scientific claim level. Included processed datasets and externally acquired raw datasets both remain evidence-limited: the package supports published response checks and catalog-audit claims under declared assumptions, not safety approval, professional certification, or complete building-code approval.
